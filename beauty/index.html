所有头部素材叙事结构高度收敛为以下 5 段。每段的"核心 Learning + 必做动作 + 避坑项"拆解为可直接复制的 SOP 表格：

段位 核心 Learning 必做动作（Checklist） 避坑项
① 钩子0 – 3s 美容美体钩子分两条线 — 男士剃须 vs 女性面部美容：
  • 剃须线：女性视角抱怨"三天两头刮胡子你不觉得很烦吗"（站女性视角让她替男性下单）
  • 抖腹/塑形线："坚持抖腹 30 天会发生什么"+ 真人腹部紫色振动器
  • 面部美容线：核桃/葡萄脸+蓝光美容仪 → 真人嫩白对比
美容美体客单 ¥361，"快速可见效果"是核心吸引力。
  • 3s 内出现 女性视角（卖男士产品也常用女性出镜）
  • 用 具象时间承诺（30 天 / 7 天 / 一次见效）
  • 用 水果可视化（核桃→葡萄）做面部老化对比
  • 开篇直接产品摆拍（无情绪）
  • 避免医疗化承诺（合规）
② 痛点3 – 10s "形象焦虑"是该品类的核心痛点：
  • 男士：胡子拉碴显老 / 油腻不修边幅
  • 女性：眼袋黑眼圈 / 面部松弛 / 双下巴
  • 夫妻互动：女主嫌弃男主形象
  • 用 家庭成员（妻子/孩子）的不满做情感杠杆
  • 突出 "中年男人秘密武器"这类话术做面子诉求
  • 过度贬损用户外貌（情感反弹）
③ 解药10 – 25s 美容美体的解药主打 "明星同款 + 雪景豪宅"：
  • 明星演员代言（康佳徐佳）正面口播
  • 雪景大平层（v01）+ 冰床产品 = 高级科技感
  • 松下/康佳等品牌信任符号全程压角
  • 必出 明星 / 真人代言镜头
  • 必出 核心卖点字幕（V 脸 / 蓝光 / 红光 / 干刮湿刮 二合一）
  • 建议加 雪景 / 大理石豪宅背景做高级感
  • 避免无品牌信任符号（让用户觉得三无产品）
④ 演示25 – 90s "前后对比 + 多场景使用"：
  • 核桃→葡萄水果对比演示（V 脸仪）
  • 男士全脸/胡须不同区域使用展示（剃须刀）
  • 女性戴红光眼罩 / 戴 V 脸仪 / 抖腹塑形多机型连放
  • 用 水果道具把抽象效果可视化（核桃皱→葡萄滑）
  • 多人群演示（男 / 女 / 老年）证明通用
  • 每个使用场景配 一句简短字幕
  • 真人对比效果过分修图（用户识破降低信任）
⑤ 收尾尾 2-5s "礼赠送男友/老公/女友 + 限时福利"：
  • 男士剃须刀：女性视角"送男友/老公"
  • 女性美容仪：自用 + 礼赠送闺蜜场景
  • 必出 礼赠场景（女主送男主 / 闺蜜互送）
  • 必出 限时优惠 / 工厂直发
  • 无礼赠场景对该品类是失分项
