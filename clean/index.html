二、跑量视频黄金五段式 SOP（核心方法论）

所有头部素材叙事结构高度收敛为以下 5 段。每段的"核心 Learning + 必做动作 + 避坑项"拆解为可直接复制的 SOP 表格：

段位 核心 Learning 必做动作（Checklist） 避坑项
① 钩子0 – 3s 清洁电器钩子的杀手锏是 "清洁结果可视化"：
  • 清洁结果型：地面光亮如镜的反光特写 → "光脚自由"
  • 极脏对比型：果汁/牛奶/油渍倾倒满地 → 一拖即净
  • 悬疑钩子型（擦窗机）："天呐天呐"+ 真人惊讶看窗外
类目均价 ¥1117 是全 7 类最高，视觉证明「值这个价」是核心。
  • 3s 内必出 "清洁前 vs 清洁后" 对比镜头
  • 用 家具反光特写（地板像镜子）做"高级感"
  • 配 大字标题"光脚自由""感谢苏泊尔"做情感代入
  • 避免单产品摆拍开场（无视觉冲击）
  • 避免功能字幕过多挤占主画面
② 痛点3 – 10s "老式打扫的狼狈"是核心痛点：
  • 夫妻吵架场景（v01）：老式扫把扫不净 → 满地纸屑+果汁 → 老婆"再拖一遍太费劲"
  • 多种污渍：奶茶 / 油 / 头发 / 宠物毛 多场景
  • 窗外脏：擦窗机配合"高楼擦窗危险"危险性
  • 用 真实污渍的极致状态（一地脏）做痛点放大
  • 结合 "夫妻吵架/家务推诿" 加生活化
  • 过度脏乱画面（审核风险）
③ 解药10 – 25s 清洁品类的转化引擎是 "高客单家居场景"：
  • 欧式家具 / 大理石地板 / 落地窗 — 让产品看起来"配得上 ¥1000+"
  • "添可极客新品真绝了" / "苏泊尔" 强品牌信号
  • 真人试用 + 一键启动 + 自动归位的"懒人友好"
  • 必出 欧式/大理石豪宅场景（高客单视觉锚定）
  • 必出 品牌 logo+品牌名口播
  • 展示 一键启动 / 自动回充 的"零门槛"
  • 避免低端家庭场景（与价格不匹配）
④ 演示25 – 90s 演示段聚焦 "清洁全场景 + 自清洁能力"：
  • 多种污渍逐一清理（果汁 / 头发 / 油渍 / 灰尘）
  • 清洁完产品自动洗刷的"机器自洁"特写
  • 地板瓷砖反光对比镜头
  • 每种污渍配 3-5 秒清洁演示 + 一句字幕
  • 必出 地板镜面反光结果验证
  • 突出 "机器自洗 / 不用洗滚刷"（用户最大痛点）
  • 过分技术参数堆砌（用户不懂转速/吸力数值）
⑤ 收尾尾 2-5s "主妇人设 + 限时福利"：
  • 家庭主妇/妈妈使用得心应手
  • 红色字幕 / 直播间引导 / 工厂特价
  • 必出 真实家庭使用场景（不是空房间）
  • 必出 价格福利信号（直降 / 满减 / 工厂直发）
  • 没有福利信号（高客单转化失败）
